Import any CSV or spreadsheet

If your collection lives in a tool without a dedicated guide — TCGplayer, Cardsphere, an old tracker, or a spreadsheet you built yourself — it still imports. The only requirement is a Name (or Card Name) column with one card per row. Every other column is optional; SpellControl recognizes the common names automatically.

Columns SpellControl recognizes

Field Accepted column names
Card name (required) Name, Card Name
Set Set, Set Code, Edition, Edition Code, Expansion, Set Name
Collector number Collector Number, Card Number, Card #
Quantity Quantity, Count, Qty
Finish Foil, Finish, Printing (values like foil, etched, true/false)
Condition Condition, Card Condition (NM, Near Mint, LP, EX, Played, Damaged…)
Language Language, Lang (full names or codes)
Price Purchase Price, Price, Price (USD), Value, Unit Price
Exact printing Scryfall ID
Folder / binder Binder Name, Folder, Category, Tags, Collection
Per-copy flags Altered, Proxy, Misprint
Rarity Rarity

Column names are case-insensitive, and column order doesn't matter.

Import into SpellControl

  1. Open SpellControl — the add-cards panel opens.
  2. Drop the file in, or paste its contents (from a spreadsheet, copy the cells — tab-separated paste works).
  3. Review the resolved cards and confirm the import.

Notes

  • Comma, tab, and semicolon delimiters are auto-detected; quoted fields (names with commas) are handled.
  • Rows with a quantity of exactly 0 are treated as unowned (wishlist entries) and skipped.
  • Name-only rows import fine — SpellControl picks a printing, and you can fix any copy's printing afterwards. Include set and collector number to pin the exact printing up front.
  • The folder/binder column is kept as each card's source category, so you can see where a copy came from.
