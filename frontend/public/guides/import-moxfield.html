Import a Moxfield collection

Moxfield's collection CSV carries edition codes and collector numbers, so SpellControl resolves each row to its exact printing — not just the card name.

Export from Moxfield

  1. On moxfield.com, open your Collection page.
  2. Open the collection menu and choose Export (CSV).
  3. Save the file — it downloads with all your copies, one row per printing.

Import into SpellControl

  1. Open SpellControl — the add-cards panel opens.
  2. Drop the CSV in, or paste its contents.
  3. Review the resolved cards and confirm the import.

The Moxfield format is auto-detected from the file's headers.

What SpellControl reads from a Moxfield export

  • Exact printing — name + edition + collector number.
  • Quantity — the Count column.
  • Finish — Moxfield's foil column, including etched; a blank value means nonfoil.
  • Condition — "Near Mint", "Lightly Played", and the rest map onto SpellControl's condition scale.
  • Language — full names like "Japanese" normalize to the standard codes.
  • Purchase price, kept on the card.

Notes

  • Rows with a count of 0 — wishlist or tradelist-only entries — are skipped rather than imported as one copy. The import summary tells you how many were skipped.
  • Moxfield deck exports (the Arena-style text) also import — see the Arena & text lists guide.
