Import a ManaBox collection

ManaBox exports are the highest-fidelity import SpellControl accepts: every row carries a Scryfall ID, so each card resolves to its exact printing with no name-matching involved.

Export from ManaBox

  1. In ManaBox, open your collection — or a single binder if you only want part of it.
  2. Tap the export/share option and choose Export → CSV.
  3. Get the file onto the device where you'll import — share it to yourself, or open SpellControl on the same phone.

Import into SpellControl

  1. Open SpellControl — the add-cards panel opens.
  2. Drop the exported file in, or paste its contents.
  3. Review the resolved cards and confirm the import.

The format is auto-detected from the file's headers — you don't tell SpellControl it's a ManaBox export.

What SpellControl reads from a ManaBox export

  • Exact printing — direct Scryfall ID lookup, no guessing.
  • Quantity per card.
  • Finish — foil, etched, or nonfoil.
  • Condition — ManaBox's near_mint / lightly_played style values are normalized to SpellControl's condition scale.
  • Language.
  • Purchase price, kept on the card.
  • Binder name — kept as each card's source category, so you can still see which ManaBox binder a copy came from. SpellControl's own binders are rule-based (color, type, set, price, tags), so your cards re-file themselves by rules rather than recreating the old folders one-to-one.

Notes

  • CSV and TSV both work — ManaBox's tab-delimited export is fine as-is.
  • Importing twice adds copies; delete an import from the import history if you need to undo one.
