Import a Deckbox inventory

Deckbox's inventory export is a plain CSV that SpellControl reads directly. Export the inventory (not the wishlist) — rows with a count of 0 are treated as unowned and skipped.

Export from Deckbox

  1. On deckbox.org, open your Inventory.
  2. Choose Export from the inventory's tools and pick CSV.
  3. Save the file.

Import into SpellControl

  1. Open SpellControl — the add-cards panel opens.
  2. Drop the CSV in, or paste its contents.
  3. Review the resolved cards and confirm the import.

The format is auto-detected from the file's headers.

What SpellControl reads from a Deckbox export

  • Printing — name + edition, plus the card number when present.
  • Quantity — the Count column; count-0 rows (wishlist or trade-only entries) are skipped rather than imported as one copy.
  • Finish — the foil column.
  • Condition — Deckbox's condition names, including "Good" and "Played", normalize to SpellControl's NM–damaged scale.
  • Language.
  • Price, when a price column is present.

Notes

  • Deckbox set names sometimes differ from official set codes; SpellControl resolves by name + set and picks the best match. Spot-check a few promos and special printings after import — you can fix any copy's printing in the app.
