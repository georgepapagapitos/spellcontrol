Import an Arena deck or plain text list

Any text list of cards imports into SpellControl — an MTG Arena export, a deck list copied from a deck site, or names you typed yourself. One card per line.

Line formats that work

1 Sol Ring (CMR) 472
1 Sol Ring (CMR)
4x Lightning Bolt
Sol Ring
  • Quantity is optional — a bare name imports one copy; 4 and 4x both work.
  • A set code in parentheses pins the printing; adding the collector number pins it exactly.
  • *FOIL*, [FOIL], *F*, *ETCHED*, and [ETCHED] after a name set the finish.
  • Section headers — Deck, Sideboard, Commander, Companion, Maybeboard — are recognized, and lines starting with // or # are ignored as comments.

Export from MTG Arena

  1. In Arena, open Decks and select a deck.
  2. Choose Export — the list is copied to your clipboard.
  3. Paste it into SpellControl.

The same works for the "export to Arena" text that deck sites like Moxfield, Archidekt, and EDHREC produce.

Import into SpellControl

  1. Open SpellControl — the add-cards panel opens.
  2. Paste the list.
  3. Review the resolved cards and confirm the import.

Notes

  • Lines without a set code resolve to a printing SpellControl picks; you can change any copy's printing afterwards. Add (SET) number to pin it up front.
  • Arena set codes occasionally differ from paper codes for remastered sets; spot-check those few cards after import.
